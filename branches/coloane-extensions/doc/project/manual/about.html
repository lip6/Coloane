1. About Coloane

1. About the Name

Coloane lies to the south of the Macao peninsula, beyond the island of Taipa.
Almost twice the size of Taipa, Coloane is an island of pine forests, eucalyptus groves, tropical vegetation and farming land.
Coloane is famed for its beautiful beaches of fine white sand.

Coloane is also the new version of graphical client for the tools-plateform FrameKit.
The previous family of clients was named Macao. Thus, now, you make the link !

2. Minimum Requirements

Coloane is packaged either into a Eclipse plugin or a standalone application.
The plugin needs :
  • A recent java installation (> 1.5.0)
  • A recent Eclipse installation (> 3.1)
The standalone application only needs a recent Java installation (> 1.5.0).

3. Download

Coloane and misc contents related to it could be find on the website : http://www.lip6.fr/coloane

4. Installation

If Coloane is packaged into a Eclipse plugin, you have to copy the jar file into the /plugin directory of your Eclipse installation. After this copy, Eclipse must be restarted. To check your installation try to find a Coloane entry in the Eclipse's plugin list (Help >> About Eclipse SDK >> Plugin Details)
In the second case, the standalone application, the directory coloane is self-content and ready to be executed. So, copy this directory and all its subdirectories into the place you dedicated to this application. Coloane is new ready to be used.
